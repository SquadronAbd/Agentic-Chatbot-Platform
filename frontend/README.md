# Frontend

React / Next.js frontend for the Agentic Chatbot Platform.

---

## Technologies

- React / Next.js
- Tailwind CSS
- Axios
- React Router

---

## Installation

```bash
npm install
```

Run

```bash
npm run dev
```

---

## Responsibilities

- Login
- Register
- Dashboard
- Chat
- File Upload
- Analytics
- Admin Panel

---

## Folder Structure

```text
frontend/
│
├── src/
├── public/
├── package.json
└── README.md
```

---

Backend API

```
http://localhost:8000
```

Swagger

```
http://localhost:8000/docs
```
```
