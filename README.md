# Agentic Chatbot Platform

An enterprise-style Multi-Agent RAG Platform built using:

- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Redis
- LangChain
- LangGraph
- Qdrant
- React / Next.js

---

## Project Structure
Agentic_Chatbot_Platform/
│
├── backend/ → FastAPI REST API
├── ai/ → Multi-Agent RAG System
├── frontend/ → React / Next.js Client
